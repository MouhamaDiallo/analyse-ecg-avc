# Analyse des troubles électrocardiographiques chez les patients atteints d’AVC

## Contexte
Projet réalisé au **Centre Muraz (Burkina Faso)** dans le cadre d’une étude sur les
accidents vasculaires cérébraux (AVC).

## Objectif
Analyser les troubles électrocardiographiques (ECG) observés chez les patients AVC
afin de mieux comprendre les profils à risque.

## Contenu du dépôt
- Rapport d’analyse (PDF)
- Résultats et interprétations statistiques

## Données
Les données utilisées sont issues de dossiers médicaux.
Pour des raisons éthiques et de confidentialité, elles ne sont pas diffusées.

## Outils
- R et Excel
